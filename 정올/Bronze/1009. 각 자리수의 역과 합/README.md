### 문제 설명

<p>양의 정수를 입력받아 역으로 보여주고 각 자리 숫자의 합을 구하는 프로그램을 작성하라.​</p>

### 입력 

 <p>21억 이하의 양의 정수를 입력받는다. 잘못된 데이터는 입력되지 않는다.</p>
 <p>하나의 결과가 나온 후에도 계속 새로운 입력을 받다가 0이 입력되면 프로그램을 종료한다.</p>
 <p>최대 10개의 양의 정수가 입력될 수 있다.</p>

### 출력 

 <p>입력받은 수의 역과 각 자리 숫자의 합을 공백으로 구분하여 출력한다.</p>
 <p>유효하지않은 "0"은 출력하지 않는다.</p>
 <p>입력받은 수의 역도 21억 이하의 정수이다.</p>
